# Kinderfest Isny Ultras 🐊

Offizielle Vereins-Website der **Kinderfest Isny Ultras** — est. 2026, 13 Mitglieder, ausverkauft.

**Live:** https://lewi1.github.io/Kinderfest_Ultras/

## Inhalt

- `index.html` — die komplette Website (eigenständig, Logo eingebettet, kein Build nötig)
- `logo.jpeg` — das Vereinswappen im Original

## Pflege

Das Kinderfest-Datum für den Countdown steht in `index.html` im `<script>`-Block ganz unten (`FEST_START` / `FEST_ENDE`). Änderungen einfach committen — GitHub Pages veröffentlicht automatisch neu.

Sämtliche Erlöse zugunsten des Kinderfests Isny.
